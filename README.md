## dialog-edit-img

基于 konva.js 的简易编辑图片弹窗组件

## 功能

-   [ ] 添加文字
-   [ ] 图片旋转
-   [x] Filters
    -   [x] blur
    -   [x] invert
    -   [x] enhance
    -   [x] grayscale
    -   [x] posterize
    -   [x] sepia
-   [ ] 输出处理
    -   [x] base64
    -   [ ] blob
    -   [ ] image
-   [ ] 配置上传处理
-   [x] 构建 npm 包

## 使用

```bash
# 安装依赖
yarn install # or npm install
yarn dev # or npm run dev
```
